# E-Commerce Delivery Performance Analysis

## 📌 Objective
Analyze delivery performance in an e-commerce dataset to identify delays, inefficiencies, and factors affecting customer experience.

## 📊 Dataset
- Source: Brazilian E-commerce Public Dataset (Olist)
- Data includes:
  - Orders
  - Customers
  - Sellers
  - Delivery timestamps
  - Geographic information

## ⚙️ Tools & Technologies
- SQL (JOIN, GROUP BY, filtering, aggregation)
- Python (Pandas, Matplotlib)

## 🔍 Methodology
1. Data extraction and transformation using SQL  
2. Combining multiple tables (orders, customers, sellers)  
3. Data cleaning and preprocessing  
4. Exploratory Data Analysis (EDA) to identify patterns  
5. Visualization of delivery performance and distribution  

## 📈 Key Findings
- Delivery time distribution shows **significant variability**, with presence of extreme delays (outliers)  
- Certain regions (e.g., São Paulo) dominate order volume and logistics flow  
- Late deliveries indicate inconsistencies in logistics performance  

## 🔑 Key Insights
- High delivery delays negatively impact customer satisfaction  
- Regional concentration suggests dependency on specific logistics hubs  
- Outliers highlight inefficiencies that require operational improvement  

## 💡 Business Impact
- Helps identify bottlenecks in delivery operations  
- Supports optimization of logistics and distribution strategy  
- Provides insights to improve **customer experience and retention**
